== Description ==

IP2Location Hello Greeting plugin displays the **Hello** greeting message in visitor's native language based on visitor's origin country.

Greet your visitors from different countries with different languages as this plugin will instantly detect the country and switch language based on visitors IP address using [IP2Location geolocation database](http://www.ip2location.com "http://www.ip2location.com").

This plugin support Chinese, English, Spanish, Arabic, Hindi, Russian, Bengali, Portuguese, Malay, French, German, Urdu, Japanese, Persian, Italian, Korean, Turkish, Vietnamese and Polish languages for display, and in case the translation is not available for a particular language, it will be substituted with English display.

This plugin supports both the **shortcode** and **widget** display. In case of any missing language that you would like to see in this plugin, please feel free to send us a request at support@ip2location.com. We will be happy to include this into the plugin.

This plugin uses IP2Location IP address geolocation BIN file for IP location lookup that free your hassle from setting up the relational database. With the monthly BIN updates, you'll always have the most up-to-date IP geolocation information and most accurate geolocation results.

IP address geolocation BIN file download: [IP2Location Commercial database](http://ip2location.com "IP2Location commercial database") | [IP2Location LITE database](http://lite.ip2location.com "IP2Location LITE database")

== Installation ==

1. Create `ip2location-hello-greeting` folder in the `/wp-content/plugins/` directory.
2. Upload `database.bin` to `/wp-content/ip2location-hello-greeting/` directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. You can now start using IP2Location Hello Greeting to customize your greeting content.

Please take note that this plugin requires minimum **PHP version 5.4**.
